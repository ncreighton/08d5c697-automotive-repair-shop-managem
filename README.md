A WordPress plugin built for independent auto repair shops, dealership service departments, and fleet maintenance operations. Features work order management with mechanic assignment, vehicle service history tracking, OEM parts inventory with reorder alerts, labor time estimate lookups using Chilton/Mitchell data structures, customer vehicle reminder emails, and a customer-facing repair status portal. Integrates with QuickBooks for accounting and Stripe for payment collection.

== Description ==

A WordPress plugin built for independent auto repair shops, dealership service departments, and fleet maintenance operations. Features work order management with mechanic assignment, vehicle service history tracking, OEM parts inventory with reorder alerts, labor time estimate lookups using Chilton/Mitchell data structures, customer vehicle reminder emails, and a customer-facing repair status portal. Integrates with QuickBooks for accounting and Stripe for payment collection.

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Automotive Repair Shop Management and Parts Inventory WordPress Plugin
